Información sobre la aplicación
  • Autor: Ricardo Moreno Cantea
  • Descripción: Esta aplicación se ha realizado para el módulo de Horas de Libre Configuración del 2º año del CFGS de Desarrollo de Aplicaciones Web (DAW). En este módulo hemos aprendido a crear aplicaciones móviles usando el framework de Ionic y, en el caso de este proyecto, se usa Firebase para poder guardar y recuperar datos de una base de datos Firestore.
  • Tecnologías utilizadas:
    • HTML 5
    • SASS/SCSS
    • TypeScript
    • Ionic
    • Angular
    • Node.js
    • Google Firebase
